<br>

<details close>
<summary><strong>Configuração do Projeto</strong></summary>

###

- *Criar projeto:*
    ```sh
    poetry add prj_agenda
    ```
    <br>

- *Definir versão Python do Projeto:*
    ```sh
    poetry env use 3.11.5
    ```
    <br>

- *Ativando ambiente virtual:*
    ```sh
    source ./.venv/bin/activate
    ```
